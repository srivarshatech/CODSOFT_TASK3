# CODSOFT TASK 3 - Simple Calculator

## Objective

Design a command-line calculator that performs basic arithmetic operations based on user input.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation
- Division-by-zero handling
- Menu-driven interface

## Technologies Used

- Python 3

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python calculator.py
```
